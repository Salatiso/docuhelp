How to Create New Templates

  1. Open an existing template (e.g., business-plan.html or business-letter.html) in your text editor.
  2. Save a copy with the new filename (e.g., novel-manuscript.html).
  3. Update the cover page title, subtitle, and document details to match the new document type.
  4. Edit the main content sections: change headings and placeholder text to suit the structure of the new document (e.g., chapters for a novel, sections for an essay).
  5. Update the Table of Contents and any page-specific instructions as needed.
  6. Test the template in your browser to ensure formatting and print-readiness.
  7. Save and add the template to this library for others to use.
